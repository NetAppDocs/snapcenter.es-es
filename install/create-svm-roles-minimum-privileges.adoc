= Cree roles de SVM con privilegios mínimos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Hay varios comandos de la CLI de ONTAP que debe ejecutar cuando crea un rol para un usuario de SVM nuevo en ONTAP. Este rol es obligatorio si configura SVM en ONTAP para su uso con SnapCenter y no desea utilizar el rol vsadmin.

* Pasos*

. En el sistema de almacenamiento, cree un rol y asigne todos los permisos al rol.
+
`security login role create –vserver <svm_name\>- role <SVM_Role_Name\> -cmddirname <permission\>`

+

NOTE: Debe repetir este comando para cada permiso.

. Cree un usuario y asigne el rol a ese usuario.
+
`security login create -user <user_name\> -vserver <svm_name\> -application ontapi -authmethod password -role <SVM_Role_Name\>`

. Desbloquee el usuario.
+
`security login unlock -user <user_name\> -vserver <svm_name\>`
